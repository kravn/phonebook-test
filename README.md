# Installation

After cloning repo, run these commands below: 

* yarn OR yarn install
* yarn json-serve
* yarn dev

# Details

Below are used to create this simple app
- NuxtJs
- Vuex
- vuex-map-fields
- json-server
- google-libphonenumber
- lodash
- moment
- vuetify

Used json-server to fully imitate API CRUD request/response behavior
